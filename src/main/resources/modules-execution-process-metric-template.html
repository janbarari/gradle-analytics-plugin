# Module Avg Duration Avg Parallel Duration Avg Parallel Rate Avg Coverage Duration Diff Rate
Avg Duration is the module median average execution duration.
Avg Parallel Duration is the module median average parallel execution duration.
Avg Parallel Rate is the module median average parallel execution rate versus Avg Duration.
Avg Coverage is the average duration rate of the module against overall build.
Duration Diff Rate is the module duration rate difference during the report period.
